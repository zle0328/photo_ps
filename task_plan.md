# 证件照生成器项目计划

## 目标

创建一个以微信小程序为主端的 uni-app 证件照生成器 MVP，包含 UI 核心链路、Tailwind 小程序样式链、激励视频解锁适配层，以及可运行的 FastAPI 图像任务后端。

## 范围

- uni-app Vue 3 + TypeScript + Vite 微信小程序工程。
- 首页、规格搜索与详情、相册选择、制作确认、结果与电子照记录等核心页面。
- 微信激励视频广告适配层；真实广告位通过环境变量配置。
- FastAPI 规格、广告奖励、图像生成任务与基础裁剪/换底色接口。
- 前后端构建、类型/静态检查和自动化测试。

## 非目标

- 本阶段不接生产 AppID、广告位、COS、数据库或生产域名。
- 本阶段不宣称 AI 抠图、发丝级人像分割或真人证件照合规质量已经完成。
- 不实现任意小游戏试玩完成回调。

## 阶段

1. 初始化前后端工程与依赖。
2. 建立设计 Token、公共组件和页面路由。
3. 实现核心 UI 和广告解锁状态机。
4. 实现后端 API 与基础图像处理。
5. 构建、测试、审计并记录真实证据。

## 验收标准

- `frontend` 可构建微信小程序产物。
- 核心页面可完成规格选择、图片选择、广告解锁、提交生成与查看结果的代码链路。
- `backend` 健康检查、规格查询、奖励会话和任务接口有自动化测试。
- 未配置真实微信/云资源时有明确开发态提示，不伪造线上验证。
